Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 

            
puts "Enter temperature in celsius:"

celsius = gets().chomp()
celsius = celsius.to_f

puts "The temp you entered is: #{celsius}"

fahrenheit = celsius * 9 / 5 + 32

puts "The temp in fahrenheit is: #{fahrenheit}"

def return_foo_bar(num)

  if num % 5 == 0

    else
      rtn_string = 'bar'
    end
  end
  if rtn_string.nil?
    rtn_string = num.to_s
  end
  return rtn_string
end
